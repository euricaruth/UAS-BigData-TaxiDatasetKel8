# UAS Big Data - Kelompok 8

## Analisis Taxi Dataset dengan Algoritma K-Means Clustering

Project ini menggunakan metode K-Means Clustering untuk mengelompokkan data perjalanan taksi berdasarkan karakteristik perjalanan.

## Tahapan Analisis

- Analisis Kualitas Data
  - Missing Value Analysis
  - Duplicate Data Checking
- Analisis Deskriptif Data
  - Statistik Deskriptif
  - Histogram
  - Boxplot
  - Scatter Plot
  - Analisis Jam Sibuk
  - Analisis Metode Pembayaran
- Data Preprocessing
  - Feature Scaling
- Clustering
  - Penentuan Jumlah Cluster dengan Elbow Method
  - K-Means Clustering
  - Evaluasi menggunakan Silhouette Score
  - Profiling dan Visualisasi Cluster

## Hasil
- Jumlah Cluster : 4
- Silhouette Score : 0.3874
